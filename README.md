# React Todo List

**TITLE: Todo List**

**Description:**
A simply TODO list built with React

**How to Run:**
-Type your task in box
-Click add to list
-Click box on task item to delete.

**Roadmap for Future Improvements**
As I continue learning React, I would love to add a previous checked list and an undo button for accidental button press.
